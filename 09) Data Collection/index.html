Data Collection

In order to model the customer arrivals and the store's operations, we chose to collect several days worth of data from the Canteen shop "Health Soup", which follows the queuing model we are interested in.

To help in our data collection, we created an interactive tool using VBA which will track the time of arrival and departures of customers and automatically calculate the interarrival times of each observation.
