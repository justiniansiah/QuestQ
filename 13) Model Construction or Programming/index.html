Model Construction/Programming


Our simulations consists of R and Jaamsim models.

The Jaamsim model was used to test our design as well as to give a rough estimate of what our results would be. The snippets of the Jaamsim model can be seen later on in the next section, Model Visualization and Animation.

We created a more in depth and rigourous model using R, and tested our values with the RStream library's Common Random Numbers feature. This allowed us to do our comparision with less varience due to the effect of random numbers.

Below is the breakdown of the code for the different phases of the simulation, with comments and descriptions to explain each part. The overarching design for the simulations are depicted previously with the IDEF0 and Functional Flow Diagram in Simulation Design, but each code has its own purposes.

The executable code are available in the folder and they run in R. Results are discussed later on in Experimental Runs.
